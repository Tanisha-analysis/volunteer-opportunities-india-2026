# Volunteer Opportunity Data Compilation - India 2026

**Task Objective:** Collect and structure information on volunteer/internship/fellowship opportunities for students in India to build research + data organization skills.

**Dataset Details:**
- **Total Records:** 16 opportunities
- **Columns:** Organization, Opportunity_Type, Eligibility, Application_Link, Deadline, Source_Date
- **Tools Used:** MS Excel, Google Sheets
- **Key Insight:** 12/16 opportunities have 'Rolling basis' deadlines
